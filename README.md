# Pyncrement
Pynput script to increment a text file number

# Prerequisites
- Install Python3
- Install pip3

# Config

- Create your increment file, and set inside config.ini
- Setup your shortcuts. The default one is CTRL+ALT+I for increment and CTRL+ALT+d for decrement.

# Running

Run pip install -r requirements.txt
Run python3 Pyncrement.py
To stop, use the exit shortcut setup in config.ini
